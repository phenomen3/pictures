#Simple web page to upload images

## Ubuntu 22.04.43 LTS

### Instalation

```
sudo apt update
```

```
sudo apt install python3-pip
```

```
sudo -H pip3 install --upgrade pip
```

```
sudo sudo -H pip3 install virtualenv
```

```
virtualenv venv
```

```
source venv/bin/activate
```

```
pip install Flask
```

```
flask --app app2.py run
```
